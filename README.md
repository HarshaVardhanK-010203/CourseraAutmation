# Coursera Automation 🎓

A Chrome extension designed to streamline Coursera course navigation and automate repetitive learning tasks through a clean and user-friendly interface.

---

## 🚀 Features

### ⚡ Media Auto-Completer

Automatically process course materials, including:

* Videos
* Readings
* Supplementary resources

### 📋 Question Extractor

Extract quiz and assignment questions into a clean, copyable format for review and study purposes.

### 🤖 Gemini AI Integration

Supports Google Gemini API integration for advanced content analysis and assistance.

### 🎯 Simple Interface

Minimal design with one-click actions and easy configuration.

---

## 📦 Installation

### Method 1: Download ZIP

1. Download this repository as a ZIP file.
2. Extract the ZIP to your computer.

### Method 2: Clone Repository

```bash
git clone https://github.com/HarshaVardhanK-010203/CourseraAutmation.git
```

---

## 🔧 Setup

1. Open Google Chrome.
2. Navigate to:

```text
chrome://extensions/
```

3. Enable **Developer Mode**.
4. Click **Load Unpacked**.
5. Select the extracted project folder.

---

## 🔑 Gemini API Setup

To use AI-powered features:

1. Visit Google AI Studio:
   https://aistudio.google.com/app/apikey

2. Create a free Gemini API Key.

3. Open the extension.

4. Paste the API key into the API field.

5. Click **Save Key**.

---

## 📖 Usage

1. Open any Coursera course.
2. Click the extension icon in the Chrome toolbar.
3. Choose the desired feature.
4. Follow the on-screen instructions.
