## ✨ Features

### 📺 Media Progress Automation

Automatically interacts with course media pages and learning materials to streamline navigation and progress tracking.

### 📋 Question Extraction

Extracts visible quiz and assignment questions into a clean, copyable format for easier review and study.

## 🚀 Installation

### 1. Download the Project

Clone the repository or download it as a ZIP file:

```bash
git clone https://github.com/your-username/coursera-auto-solver.git
```

### 2. Load the Extension

1. Open Chrome.
2. Navigate to `chrome://extensions/`
3. Enable **Developer Mode**.
4. Click **Load Unpacked**.
5. Select the project folder.

---

## 🔧 Configuration

1. Open a supported Coursera course page.
2. Click the extension icon from the Chrome toolbar.
3. Configure the available options.
4. If using Gemini features, obtain an API key from Google AI Studio and save it in the extension settings.
